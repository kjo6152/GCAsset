AssetBundle을 생성해주는 클래스

Unity 에디터에서 사용할 수 있는 메뉴를 추가해준다. 실제 프로그램에서는 컴파일되지 않으며 데이터에서 리소스 폴더를 우클릭하여 "Build Resources AssetBundle"를 클릭하면 AssetBundle을 생성한다. 생성된 AssetBundle은 GCAsset 내부에 저장되며 컨트롤러와 연결 시 전송되어 사용할 수 있다.

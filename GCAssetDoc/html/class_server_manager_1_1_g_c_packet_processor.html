컨트롤러에게서 패킷을 받아 처리하는 클래스

각 컨트롤러마다 할당되며 실제 통신이 일어나는 클래스이다. 패킷을 분석하여 그에 알맞는 처리를 하거나 다른 객체로 전달해준다. 해당 컨트롤러에게 이벤트를 전달해준다.

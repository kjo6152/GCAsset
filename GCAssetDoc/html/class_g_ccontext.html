상세한 설명

GCAsset을 이용하기 위한 여러 객체 및 리소스를 관리하는 클래스

GCAsset의 기능을 활용하기 위한 모든 객체와 정보들을 가지고 있는 최상위 클래스이다. 싱글톤 패턴으로 하나의 객체만 유지되며 객체 생성시 씬에 자동으로 오브젝트를 생성하여 동작한다. 씬이 변경되어도 해당 오브젝트는 파괴되지 않고 계속 유지된다. 객체 생성시 아래 세 클래스의 객체를 생성한다.

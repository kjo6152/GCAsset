상세한 설명

GCAsset와 컨트롤러간 주고 받을 리소스를 관리하는 클래스

사용자가 생성한 AssetBundle을 컨트롤러로 전송할 수 있도록 xml을 파싱하여 정보를 가지고 있는 클래스이다. AssetBundle에 대한 경로나 xml 문서의 명칭 등 상수를 가지고 있다.
